Cube Racing is a 3D endless racing game developed in Unity where the player controls a cube navigating through a track filled with obstacles and collectibles. The game is designed with progressive difficulty, an engaging scoring system, and a smooth game flow that keeps the player challenged at every stage.
The player cube moves forward automatically using Rigidbody physics force, while the player controls left and right movement using arrow keys or A/D keys. Boundary clamping ensures the cube stays within the track limits at all times. The camera follows the cube along the Z axis with a defined offset, providing a clean and consistent view throughout the game.
The scoring system rewards players with points for collecting items tagged as collectibles. Every item collected adds 0.5 to the score, which is displayed live on screen during gameplay and shown again on the game over screen as the final score.
The game features five progressive levels unlocked at score thresholds of 20, 40, 60, and 80. When a threshold is crossed, a Level Up panel appears for two seconds, the player speed increases, and a completely new scene is loaded representing the next level. This keeps the experience fresh and increasingly challenging as the player progresses.
A countdown timer adds pressure to each level, starting at 30 seconds. When time runs low, the timer text turns orange as a warning. Reaching zero triggers a game over. However, leveling up rewards the player with 10 bonus seconds, balancing difficulty with progression.
Game flow is managed through a clean UI system including a tap to start screen, a game over panel, a restart button that reloads Level 1, and a quit button. Audio feedback is provided for both collectible pickups and obstacle collisions, enhancing the overall player experience.
